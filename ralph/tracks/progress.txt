## Codebase Patterns
- Static HTML site: no build step, Tailwind via CDN, vanilla JS only
- All interactivity in assets/js/main.js using IIFE modules and data attributes
- Brand colors: navy #1A2E4A, amber #C8963E, cream #F7F4EF, slate #64748B
- Fonts: Playfair Display (headings), Inter (body) — loaded via Google Fonts CDN
- Tailwind config is inline in each HTML file's <script> tag (not a separate config file)
- Placeholder images use placehold.co with brand colors
- All inter-page links use relative paths (e.g., ./about.html, not /about.html)
- Nav id="main-nav", toggle btn id="nav-toggle", mobile menu id="nav-mobile-menu"
- Contact form id="contact-form" with field ids: field-name, field-email, field-description
- Lightbox div id="lightbox", image id="lb-image", etc.
- Nav: fixed top, z-50, bg-navy always (not transparent — scroll effect adds shadow via JS)
- Active nav link: text-amber + border-b-2 border-amber; inactive: text-white hover:text-amber
- Footer: bg-navy, 3-col md:grid-cols-3, stacks to 1 col on mobile
- No typecheck tooling — quality check is well-formed HTML structure
